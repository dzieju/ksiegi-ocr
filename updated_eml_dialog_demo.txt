NEW FEATURE: Save File Option
=============================
The new "💾 Zapisz plik" option allows users to:

1. Save EML files to any location on their computer
2. Choose a custom filename (defaults to email_YYYYMMDD_HHMMSS.eml)
3. Preserve the original EML format for compatibility
4. Use the saved file later with any email application

Integration:
- Works seamlessly with mail search results
- Available in all contexts where EML opening is used
- Maintains compatibility with existing workflow
- No GUI changes required for existing users
