⚙️ How It Works

🔍 PDF Search Process:

  1. User enters search text and optionally enables "Pomijaj już wyszukane PDF-y"
  2. For each PDF attachment:
    • Generate unique identifier (name + SHA256 content hash)
    • Check if PDF was already searched with this search term
    • If skip option enabled and PDF already searched → mark as skipped
    • Otherwise → perform normal PDF search and mark as searched
  3. History tracking:
    • Track searched PDFs with timestamps and match results
    • Track skipped PDFs with skip counts
    • Save history to pdf_search_history.json
